# Changelog

All notable changes to Email Guard will be documented in this file.

## [1.0.0] - 2024-10-24

### Added
- Initial release with full feature set
- Expected Sender & Link Domain List functionality
- Real-world sender verification
- Advanced link analysis with homograph detection
- Whitelist override & user feedback loop
- Privacy-first local analysis with optional enhanced mode
- Organization mode with reporting dashboard
- Explainable alerts with severity levels
- Cross-client support (Gmail and Outlook Web)
- Offline capability for core features
- Comprehensive documentation

### Security
- Privacy-first design
- Local-only processing by default
- No email content storage
- Encrypted settings storage
- Minimal permission requests
